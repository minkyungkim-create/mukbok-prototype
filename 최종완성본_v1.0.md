## ✨ 주요 기능

### 1. 온도 클릭 → 테마 변경
```
사용자가 온도 클릭 (34°C → 20°C)
  ↓
theme = '치맥의 계절' (cold)
  ↓
탭 자동 정렬 (🍗 치맥이 맨 앞)
  ↓
브랜드 & 상품 자동 로드
```

### 2. 탭 클릭 → 콘텐츠 업데이트
```
사용자가 탭 클릭 (수박 탭)
  ↓
쿠폰 정보 업데이트
  ↓
브랜드 아이콘 변경
  ↓
상품 카드 업데이트
```

### 3. 온도별 추천 메뉴 자동 정렬
```
초기: 🧊 빙수 | 🍉 수박 | 🔥 마라 | ...
온도 변경 (28°C):
최종: 🧊 빙수 | 🍉 수박 | ... (빙수가 추천이므로 유지)
```
